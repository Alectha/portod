# Image Naming Convention

Use this format for local project images:

`area-purpose-variant.ext`

Rules:

- Use lowercase only.
- Use kebab-case, not spaces or underscores.
- Start with the section or area, such as `hero`, `about`, or `achievement`.
- Use a short purpose label, such as `portrait`, `illustration`, `certificate`.
- Add a variant only when needed, such as `main`, `preview`, or `dark`.

Examples:

- `hero-abstract-illustration.svg`
- `about-profile-portrait.svg`
- `achievement-certificate-preview.svg`
